# Module 1 - Create a Kubernetes cluster

## Useful commands:

### check `minikube` installation
```shell
$ minikube version
```

### start the cluster
```shell
$ minikube start
```

### check CLI installation
```shell 
$ kubectl version
```

### cluster details
```shell
$ kubectl cluster-info
```

### view nodes in the cluster
```shell
$ kubectl get-nodes
```
